# naver_movie_crawling

## Dependency
 - selenium
 - beautifulsoup4
 
## Crawling Target
 - 영화제목
 - 영화 카테고리 (ex: 멜로, 액션, SF, ...)
 - 국가 (한국, 영국, 미국, ...)
 - 관객 평점
 - 네티즌 평점
 - 줄거리 요약문
 - 줄거리 본문
 - 인기 리뷰 (추천 수가 제일 높은 댓글)
